### 🚀 Em destaque

**Sistema de gestão de expedição** — sistema completo que desenvolvi e mantenho em produção em um e-commerce multicanal.

- **Integração multicanal via BaseLinker** — sincronização de pedidos, estoque e notas entre marketplaces e a operação.
- **Módulo WMS de endereçamento fixo** — controle de estoque por endereço no armazém, com fluxo de bipagem para conferência e estocagem.
- **Estação de devoluções** — em produção, com login por perfil de usuário, sincronização diária automatizada e inicialização automática.
- **Serviço de impressão de etiquetas térmicas** — agente HTTP local em Python conversando com impressora térmica via protocolo TSPL.
- **Automação e continuidade** — rotinas agendadas, backup de banco e integrações com notificações.
